# MyPlace

## Description  
A simple front-end project built with HTML, CSS, and JavaScript, including a landing page, signup page, and dashboard.

## Technologies  
- HTML  
- CSS  
- JavaScript  

## How to Run  
```bash
git clone https://github.com/imenmakhbouche/myplace.git
cd myplace
# Open index.html in your browser
```

## Files  
- `index.html` — landing page  
- `signup.html` — sign-up page  
- `dashboard.html` — dashboard  
- `style.css` & `dashboard.css` — styles  
- `main.js` — scripts
